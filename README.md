# 🚗 Electric Vehicle Data Analysis

## Overview
This project analyzes Electric Vehicle (EV) data using Python to identify trends, compare manufacturers, detect outliers, perform statistical analysis, and recommend suitable electric vehicles.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook

## Project Tasks
- Data Cleaning
- Manufacturer Analysis
- Outlier Detection
- Battery Capacity vs Driving Range Analysis
- EV Recommendation System
- Hypothesis Testing (Tesla vs Audi)

## Files
- Electric_Vehicle_Analysis.ipynb
- Electric_Vehicle_Data_Analysis.pptx
- Dataset

## Key Insights
- Compared battery capacities across manufacturers.
- Identified energy consumption outliers.
- Found a positive relationship between battery capacity and driving range.
- Built an EV recommendation system.
- Performed statistical testing to compare manufacturers.
